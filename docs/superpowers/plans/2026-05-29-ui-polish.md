# UI Polish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Unify the UI styling of `UserPreferences.vue`, `DpiSettings.vue`, and `MacroSettings.vue` to a "Sleek & Minimal" design system based on predefined CSS variables.

**Architecture:** Replace hardcoded colors (slate/gray) with theme-driven CSS variables (`--bg-card`, `--border-card`, `--text-primary`, `--shark-primary`). Ensure consistent padding, rounded corners, and input styling across all tabs.

**Tech Stack:** Vue 3 (Composition API), Tailwind CSS.

---
